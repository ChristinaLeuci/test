VID
===

A web site and CMS for [Visual Image Display](http://www.visualimagedisplay.com/home/).  

### Installation

```shell
$ git clone git@github.com:nporteschaikin/VID
$ cd VID
$ bundle install
$ rake db:setup
```

### Usage

```shell
$ rails server
```
